# Repository Instructions

## Commit Messages

When the user asks to commit changes in this repository, follow [.agent/workflows/commit.md](.agent/workflows/commit.md) and use the commit style in [.agent/rules/commit-style.md](.agent/rules/commit-style.md).

Default to area prefix commits:

```text
<Area>: <Summary>
```

Choose the narrowest accurate area. Stage only files that belong to the requested change, preserve unrelated user changes, and include a commit body when the motivation, validation, or safety impact is not obvious from the summary.
